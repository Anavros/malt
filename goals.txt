
Dead-Simple Custom Language Parser:
- Extremely lightweight and easy to use.
- Only one file.
- Only a few public functions.
- Less powerful than other configurators.
- Extremely simple syntax.

Custom Langage Restrictions:
- Each statement is seperated by newlines.
- No multi-line statements.
- Each line starts with a single keyword.
- Everything after the keyword is unordered.
- Each word is a value.
- There are no keys.
- Values must be unique enough to be recognized automatically.

Bonus Goals:
- Code generator for vim syntax highlighting.
